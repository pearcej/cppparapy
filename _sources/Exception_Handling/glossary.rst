Glosario
--------

.. glossary::
    excepción
        es una respuesta a una circunstancia inusual mientras un programa se ejecuta.

    error lógico
        un error con el cual el programa/código funciona, pero funciona incorrectamente.

    error de tiempo de ejecución
        un error que ocurre cuando se ejecuta un programa.
        
    error sintáctico
        un error en la estructura de la declaración o expresión del código.
